# AI2NAMD
AI2NAMD uses graph neural network to realize Hamiltonian prediction of excited state carrier relaxation dynamics.
All the examples in the code are based on Si quantum dots. Due to the complexity of the Hamiltonian data, we only provide some of the examples.A full description an analysis can be found in our paper.(https://doi.org/10.1021/acs.jctc.5c02178)
